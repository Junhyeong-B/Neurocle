# 벡터(SVG) 기반의 드로잉 툴 구현

## 📌 프로젝트 구성

- Project: `create-react-app`, `Typescript`
- Convention: `ESLint`, `Prettier`
- Style: `module CSS`
- Library: `konva`, `react-konva`, `react-svg-path`

<br />

## 🧑🏻‍💻 실행 방법

```
$yarn
$yarn start
```
